SPA (Single Page Application) desarrollada en React 19 con Vite para consumir la API de catálogo de productos.

## Tecnologías
- React 19
- Vite
- Bootstrap 5
- Fetch API

## Requisitos previos

- Node 20 o superior

## Ejecutar

```bash
# Instalar dependencias
npm install

# Ejecutar la app
npm run dev
